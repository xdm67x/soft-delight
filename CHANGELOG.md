# Changelog

All notable changes to the Soft DeLight extension will be documented in this file.

## [1.0.0] - 2024-03-28

### Added
- Initial release
- Light theme with soft, comfortable colors
- Dark Edition theme with warm navy tones
- Custom icon for the extension
- Comprehensive README with color palette documentation

### Theme Features
- Off-white background (`#FAF9F6`) for light theme
- Deep navy background (`#1A1A2E`) for dark theme
- No harsh reds - replaced with warm oranges and indigos
- Optimized syntax highlighting for TypeScript, JavaScript, Python, Rust
- Carefully tuned token colors for maximum readability
